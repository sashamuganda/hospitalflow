## 2024-05-22 - Accessibility and Haptic Standards for Custom Components | **Learning:** | Custom interactive components in Flutter (e.g., those using GestureDetector) lack semantic meaning for assistive technologies and tactile feedback by default. | **Action:** | Always wrap custom buttons in a Semantics widget with `button: true` and a label, and trigger `HapticFeedback.lightImpact()` on tap to ensure a standard and pleasant user experience.
